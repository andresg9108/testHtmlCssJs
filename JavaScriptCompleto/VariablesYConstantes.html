Variables y constantes

Nota: JavaScript es «Case Sensitive», es decir, sensible a mayúsculas y minúsculas.

    // camelCase: miVariable
    var miVariable = 'Hola'
    const miConst = 16
    let miLet = true

var vs. let

    var name = "Andrés"
    var name = "Juan" // Con var la variable se puede declarar dos veces

    let last_name = "González"
    last_name = "Garcia" // Con let la variable no se puede declarar dos veces

Nota: let es más restringido que var en muchos contextos como los ciclos, las funciones y los condicionales.

const

    const MI_CONSTANTE = "Hello World"
    const PI = 3.14

Nota: Si asignamos una constante con un arreglo, función o objeto, no podemos reasignarlo, pero sí podremos cambiar su contenido.
