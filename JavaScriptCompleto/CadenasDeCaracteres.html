Cadenas de caracteres

Comillas

    var miCadena = "Escapar comillas: \"Hola Mundo\""
    miCadena = 'Secuencias de escape: "Hola Mundo"'
    miCadena = "Otra secuencias de escape: 'Hola Mundo'"

Secuencias de escape

  • \' - Comilla Simple
  • \" - Comilla Doble
  • \\ - Barra invertida
  • \n - Línea Nueva
  • \r - Retorno de Carro
  • \t - Tabulación
  • \b - Retroceso
  • \f - Salto de Página
    console.log("\\ Hola Mundo \\ \nDesarrollador: \"Andrés González\" \n\tHTML\n\tCSS\n\tJavaScript")

Concatenación de cadenas de caracteres

    const nombre = "Andrés"
    const apellido = "González"
    var cadena = nombre + " " + apellido
    cadena = `${nombre} ${apellido}` // Plantillas literales 
    cadena = nombre
    cadena += " " + apellido

Operaciones con cadenas de caracteres

    const cadena = "Hola Mundo"
    const tamanoDeCadena = cadena.length
    var caracter = cadena[0] // Primer carácter 
    caracter = cadena[2] // Tercer carácter
    caracter = cadena[10] // undefined
    caracter = cadena[tamanoDeCadena-1] // Último carácter

Nota: Las cadenas de caracteres son inmutables por lo tanto no de puede hacer algo como cadena[0] = “H”
